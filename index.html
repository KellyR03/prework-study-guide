HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • The title element is the title for the webpage.
  • The html opening tag represents the start of the html document.
  • The script element embeds client side scripts in the html document.
  • Html attributes are always defined in the beginning element, never the closing.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • External css style sheet allows developers to keep all their css rules in a sep file
  • Internal style sheet is used to identify elements
  • The img selector assigns property values to any image element in the markup
  • The h1 and h2 rule assignment is a postioning property

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch: creates a new branch and switches to it

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • A function is simply a set of instructions that tells the computer how to perform a certain task.
  • An array is a single variable that is used to hold a group of data.
  • control flow is the order in which a computer executes code in a script.
  • Javascript is a high level dynamic programming language that allows users to interact with websites.
  • We use the var keyword to declare a variable.
